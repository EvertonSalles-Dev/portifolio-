# 🚀 Everton Salles — Portfólio v2

> Portfólio cinematográfico com hero em vídeo, estilo HUD militar e design dark mode premium.

---

## 🎯 Sobre o Projeto

Site de portfólio pessoal com visual inspirado em interfaces aeronáuticas (HUD).
Desenvolvido com HTML, CSS puro e JavaScript vanilla — sem frameworks, leve e rápido.

---

## ✨ Funcionalidades

- ✅ Hero com vídeo em background fullscreen
- ✅ Interface estilo HUD com cantos decorativos e dados de telemetria
- ✅ Animações de entrada suaves com `IntersectionObserver`
- ✅ Barras de skill animadas ao entrar na viewport
- ✅ Cards de projetos com efeito hover + gradiente no topo
- ✅ Formulário de contato integrado
- ✅ Layout responsivo (mobile e desktop)
- ✅ Scroll suave entre seções
